# pontential-crud
Potencial para um crud

# Descrição
API JSON REST utilizando o framework NestJS, utilizando os métodos (​GET​, ​POST​, ​PUT​, DELETE​).

# Instalação
```
$ docker-compose up
```
```
$ npm install
```
```
$ npm run start
```

# Teste
```
$ npm run test
```

# URL Swagger com todos os endpoints
```
http://localhost:3000/api
```
